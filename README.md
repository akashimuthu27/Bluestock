## Getting Started

### Prerequisites
- Python 3.8+
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/akashimuthu27/Bluestock.git
   cd Bluestock
   ```
2. Navigate to the specific day's subdirectory and follow its local instruction guide:
   ```bash
   cd "Day 1"
   pip install -r requirements.txt
   ```
